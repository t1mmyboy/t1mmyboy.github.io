Python

Bij Da Vinci hebben we tot nu toe in Python het gehad over Variabelen (het opslaan van data), Input (je krijgt een vraag en het maakt een variabel aan met wat je invult.) Lists (verzamelingen van Variabelen), Dictionaries (meerdere stukjes data opslaan als een pair bijvoorbeeld X hoort bij Y) en Functions (stukjes code dat je kan opslaan en later oproepen) en van alles daartussen.
We hebben dus ook opdrachten gemaakt. Bij één van mijn favorieten opdrachten moest je een game maken in Python, dat was toen we net met input hadden gewerkt, dus was het nog steeds wel nieuw maar het was een hele leuke opdracht omdat je de vrijheid had om te doen wat je wou maken en het daarna ook nog realiseren. Bij een andere leuke opdracht kreeg je een programma waar je lege Functions had en je die moest invullen. Ik vond dat een leuke opdracht omdat het lastig was omdat het maar op weinig manieren werkte, dus voelde het heel belonend als je een opdracht goed maakt

HTML & CSS

Ook hebben we HTML en CSS gehad, dat is meer de pagina’s maken die je op het internet ziet, dat is ook nieuwer voor ons dus zijn we nog niet heel ver. We hebben tot nu toe geleerd over hoe je tekst en kleuren kan toevoegen aan een pagina, hoe je een pagina kan veranderen gebaseerd op hoe groot je scherm is (Bijvoorbeeld op een computer tekst naast elkaar zetten maar op je telefoon alles onder elkaar)
Bij een andere opdracht voor HTML en CSS moest je zorgen dat wanneer je de grootte van je scherm veranderd, de kleur van de pagina mee veranderd. Ik vond dit ook een leuke opdracht omdat je zelf moest opzoeken en uitzoeken hoe het moest. Hiervan heb ik ook een voorbeeld.
